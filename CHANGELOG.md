# Changelog

All notable changes to Travallee will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Admin Frontend dashboard with Tailwind CSS styling
- User authentication system with localStorage persistence
- Demo credentials for testing
- Comprehensive component library (Button, Badge, Card, Modal, etc.)
- Sample data for guests, rooms, deals, vendors, and reviews
- React Router v6 navigation setup
- Custom hooks: useAuth, useModal, usePagination

### Fixed
- Import/export type-only declarations for TypeScript compliance
- Tailwind CSS v3 compatibility (gradient classes)
- Route structure for proper page rendering

## [0.1.0] - 2024-03-21

### Added
- Initial project setup with Docker support
- Admin Frontend (React + TypeScript + Vite)
- Travallee Mobile App (React Native + Expo)
- Backend services scaffold (Microservices architecture)
- User Frontend starter
- Basic docker-compose configuration

### Changed
- Project structure organized into separate frontend and backend packages

### Documentation
- Added README files for each sub-project
- Setup instructions for development
